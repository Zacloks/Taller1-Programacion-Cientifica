## Descripción Proyecto

Este proyecto modela la red de artículos de Wikipedia como un **grafo dirigido**, donde cada nodo representa un artículo y cada arista un enlace entre artículos. Sobre esta estructura se calculan métricas estructurales, se implementan algoritmos de recorrido (BFS y DFS) y una versión simplificada de **PageRank** para identificar los artículos más relevantes de la red.

El sistema está desarrollado en Python con programación orientada a objetos, sin uso de librerías de grafos externas.

## Dataset

Se utiliza el dataset **Wiki-Topcats**, disponible en [Kaggle](https://www.kaggle.com/datasets/wolfram77/graphs-snap-wiki).

| Archivo | Descripción |
|---|---|
| wiki-topcats.mtx | Grafo principal de enlaces entre artículos |
| wiki-topcats_Categories.mtx | Relación entre artículos y categorías |
| wiki-topcats_Category_names.txt | Nombres de las categorías |
| wiki-topcats_pagenames.txt  | Nombres de los artículos |
